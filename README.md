# TextEditor

TextEditor is a computer program that edits plain text written in Java. 

## Installation

Once you cloned the git repository, all you have to do it is type in your terminal:

```bash
java Editor.java
```

## Usage
All you have to do is to to execute it:
```python
java Editor.java
```
